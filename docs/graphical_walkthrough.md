# Graphical walkthrough of the Portal

## Authentication and Registry
**Login Gateway:**
The login page utilizes a centered card layout with a professional color palette. Upon authentication, users are routed to their respective dashboards.
![Login Screen Placeholder](public/login-placeholder.jpg)

**Registry Dashboard:**
The Registry Dashboard displays real-time enrollment statistics and provides quick-action buttons for onboarding students, lecturers, and parents.
![Registry Dashboard Placeholder](public/registry-dashboard-placeholder.jpg)

**Student Registry:**
Implemented as a robust, searchable data grid allowing Registry staff to manage and register new students.
![Students Registry Placeholder](public/registry-students-placeholder.jpg)

## Academic Management
**Lecturer Interface - Dashboard:**
Lecturers are provided with a streamlined overview to minimize administrative overhead and surface pending alerts.
![Lecturer Dashboard Placeholder](public/lecturer-dashboard-placeholder.jpg)

**Lecturer Interface - Grades Management:**
A dedicated grading interface allowing lecturers to efficiently log, convert, and sync student scores to the database.
![Lecturer Grades Management Placeholder](public/lecturer-grades-placeholder.jpg)

**Student Self-Service Portal - Dashboard:**
Students access a main dashboard featuring live widgets that display their Current CGPA, Attendance Rate, and unread messages.
![Student Self-Service Dashboard Placeholder](public/student-dashboard-placeholder.jpg)

**Student Self-Service Portal - Academic History:**
Functioning as a digital transcript, this page uses accordion UI elements to group historical data by academic level, providing a clear breakdown of credit units and cumulative GPA.
![Student Academic History Placeholder](public/student-academics-placeholder.jpg)

## Parent Oversight
**Parent Dashboard:**
Parents are provided with an interface that surfaces their linked children's critical academic vitals and discipline history.
![Parent Dashboard Placeholder](public/parent-dashboard-placeholder.jpg)

**Parent-Teacher Messaging:**
A dedicated Messaging tab provides a chat-like interface allowing parents to maintain direct dialogue with course instructors.
![Parent-Teacher Messaging Placeholder](public/parent-messaging-placeholder.jpg)
